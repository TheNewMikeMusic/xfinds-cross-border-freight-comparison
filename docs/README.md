# 文档目录

## 📚 文档结构

### 部署文档 (`deployment/`)
- `DEPLOY.md` - 主要部署指南（保留在根目录以便快速访问）
- `服务器更新指令.md` - 服务器更新快速参考
- `部署成功总结.md` - 部署完成后的总结文档
- `部署验证清单.md` - 部署验证检查清单

### 故障排查 (`troubleshooting/`)
- `修复HTTPS-443端口.md` - HTTPS 443 端口问题排查
- `修复应用构建问题.md` - 应用构建问题排查
- `验证HTTPS.md` - HTTPS 验证指南

### 项目文档
- `README.md` - 项目主文档（根目录）
- `GPT5_CODEX_PROMPT.md` - GPT5 代码提示文档
- `UI_OPTIMIZATION_PROMPT.md` - UI 优化提示文档

## 🚀 快速开始

1. **部署应用**: 查看根目录的 `DEPLOY.md`
2. **遇到问题**: 查看 `troubleshooting/` 目录下的相关文档
3. **开发指南**: 查看根目录的 `README.md`
